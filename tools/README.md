# Tools Directory

This directory contains automation scripts and utilities for the VIANEO framework.

## Structure

### `/python/`
Python scripts for:
- DOCX document generation
- Data formatting and validation
- Template processing
- Export utilities

## Tool Types

### Document Generators
Scripts that automate the creation of formatted DOCX documents from data:
- Executive briefs
- Assessment reports
- Diagnostic comments
- Professional deliverables

### Data Processors
Utilities for:
- Parsing application materials
- Extracting structured data
- Validating evidence citations
- Calculating dimension scores

### Example Data
Sample data files demonstrating:
- Input formats
- Expected outputs
- Data structures
- Validation criteria

## Usage

Tools are designed to:
- Accelerate evaluation workflows
- Ensure consistent formatting
- Automate repetitive tasks
- Validate data quality

Scripts and examples will be populated from the vianeo-platform-tools repository.

## Requirements

Python tools require:
- Python 3.7+
- python-docx library
- Additional dependencies listed in requirements.txt (to be added)
