## COMMON FIXES (2-minute corrections)

**Bullet >60 characters?**
Shortening techniques:
- Remove adjectives: "very", "really", "extremely"
- Use numbers: "15 hours" not "significant time"
- Use symbols: "&" not "and", "/" not "or"
- Abbreviate: "%" not "percent", "#" not "number"
- Remove articles: "the", "a", "an"

**Example:**
- ‚ùå (72 chars) "Reduce the amount of time spent on manual administrative tasks daily"
- ‚úÖ (54 chars) "Reduce time on manual admin tasks by 50%+ daily"

**Age is a range?**
- ‚ùå "35-45 years old"
- ‚úÖ "42" (pick representative age)

**Overlapping personas?**
- Test: Would both buy the same product for same reason?
- If YES ‚Üí Merge or find what makes them distinct
- Focus on behavioral differences, not just titles

**Current solutions vague?**
- ‚ùå "Uses spreadsheets and various tools"
- ‚úÖ "Uses Excel ($0/mo) with manual data entry, Slack ($12/user/mo) for team updates"
