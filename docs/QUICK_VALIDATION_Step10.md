## COMMON FIXES (2-minute corrections)

**Too many sentences (9+)?**
- Combine related ideas with commas
- Remove redundant statements
- Each sentence must add NEW information

**Missing owners?**
- Add after action: "(Owner Name)"
- Format: "Shawna should [action] (Shawna)"
- Or: "Leadership team must [action] (Shawna, Soufiane)"

**Em dashes present?**
- Replace "‚Äî" with ", " (comma + space)
- Or use parentheses: (like this)
- Never use em dashes in VIANEO outputs

**Vague language?**
- Replace "many" ‚Üí actual number
- Replace "some concerns" ‚Üí specific vulnerability
- Replace "needs work" ‚Üí "scored 1.8, requires X"
- Replace "could" ‚Üí "must" or "should"

**Generic statements?**
- Add specifics: "50+ beta users", "zero revenue"
- Add names: "Shawna", not "the founder"
- Add numbers: "$20-50 ARPU", not "pricing model"
