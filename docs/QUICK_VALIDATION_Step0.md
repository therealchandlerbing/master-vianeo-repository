## COMMON FIXES (2-minute corrections)

**Over character limit?**
- Remove filler words: "very", "really", "actually", "basically"
- Use symbols: "&" instead of "and", "%" instead of "percent"
- Remove redundant phrases

**Solution in problem (B2)?**
- Rewrite from user perspective: pain felt, not solution offered
- Test: Remove your company name - problem still exists?

**Vague claims?**
- Replace "many" ‚Üí actual number
- Replace "significant" ‚Üí percentage/amount
- Replace "growing" ‚Üí specific growth rate
