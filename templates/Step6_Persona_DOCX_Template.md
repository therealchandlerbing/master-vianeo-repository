# Step 6: Persona DOCX Format Specification

**Purpose:** Professional Word document template for evidence-based user personas
**Output:** Committee-ready persona analysis (3-11 pages)
**Format:** Microsoft Word (.docx)

---

## Document Structure

### Total Length: 3-11 Pages

| Section | Pages | Content |
|---------|-------|---------|
| Cover Page | 1 | Title, validation summary, metadata |
| Evidence Base | 1 | Research overview, validation breakdown |
| Individual Personas | 1-2 each | 3-5 personas, each on new page |
| Comparison Matrix | 0.5 | Optional summary table |

---

## Page 1: Cover Page

### Layout

```
[COMPANY LOGO - if available]

[Company Name]
Vianeo Business Model Evaluation

USER PERSONA ANALYSIS

[Project subtitle or domain description]

────────────────────────────────────────

Prepared: [Month Year]
Validation Status: [X] personas based on [Y] total interviews
Evaluation Phase: Desirability Assessment

VIANEO Framework v2.0
```

## Pages 3+: Individual Persona Pages

### Persona Page Layout

Each persona occupies 1-2 pages with this structure:

#### Header Section

```
┌─────────────────────────────────────────────────────────────┐
│  PERSONA [NUMBER]: [FIRST NAME]                              │
│  [Validation Badge - colored box]                            │
└─────────────────────────────────────────────────────────────┘
```

#### Validation Badge Styles

**✓ VALIDATED (5+ interviews)**
```
┌────────────────────────────────────────────────────┐
│ ✓ VALIDATED                                        │
│ [X] interviews conducted [Month-Month Year]        │
└────────────────────────────────────────────────────┘
Background: #E8F4EA | Text: #2D7A3E | Border: #2D7A3E
```

**⚠ PARTIALLY VALIDATED (1-4 interviews)**
```
┌────────────────────────────────────────────────────┐
│ ⚠ PARTIALLY VALIDATED                              │
│ [X] interviews, [Y] additional recommended         │
└────────────────────────────────────────────────────┘
Background: #FFF4E6 | Text: #CC7A00 | Border: #CC7A00
```

**✗ NEEDS VALIDATION (0 interviews)**
```
┌────────────────────────────────────────────────────┐
│ ✗ NEEDS VALIDATION                                 │
│ 0 interviews, [X-Y] needed for [segment]           │
└────────────────────────────────────────────────────┘
Background: #FFEBEE | Text: #C62828 | Border: #C62828
```

### Persona Content Sections

#### Section A: Requester Profile

**Layout:** Two-column table

| Left Column (40%) | Right Column (60%) |
|-------------------|-------------------|
| **First Name:** | [Name] |
| **Age:** | [Number] |

**Life & Motivations:**
[2-3 sentences - 11pt Calibri, justified]

**Personality & Values:**
[2-3 sentences - 11pt Calibri, justified]

#### Section B: Field of Application

**Layout:** 2x2 Grid with callout boxes

```
┌─────────────────────────┐  ┌─────────────────────────┐
│ THINKS & FEELS          │  │ OBSERVES                │
│ [2-3 sentences]         │  │ [2-3 sentences]         │
└─────────────────────────┘  └─────────────────────────┘

┌─────────────────────────┐  ┌─────────────────────────┐
│ DOES                    │  │ OTHERS SAY              │
│ [2-3 sentences]         │  │ [2-3 sentences]         │
└─────────────────────────┘  └─────────────────────────┘
```

**Box Styling:**
- Background: #f7fafc
- Border: 1pt solid #e2e8f0
- Padding: 8pt
- Header: 10pt Bold, #2E5C8A

#### Section C: Activities & Challenges

**Layout:** Three-column format

| Tasks | Pains | Expectations |
|-------|-------|--------------|
| Header: #10b981 | Header: #ef4444 | Header: #3b82f6 |

**Bullet Styling:**
- Font: 10pt Calibri
- Max 60 characters per bullet (CRITICAL)
- 4-6 bullets per column
- Custom bullet icons by category

#### Section D: Current Solutions

**Header:** "Current Solutions" (12pt Bold, #2E5C8A)

[2-3 sentences describing current tools with gaps]

**Bullet list of specific tools:**
- **[Tool Name]:** [Limitation] - 10pt Calibri

---

## Character Limits (STRICTLY ENFORCED)

| Field | Limit | Notes |
|-------|-------|-------|
| Tasks bullets | 60 chars | Each individual bullet |
| Pains bullets | 60 chars | Each individual bullet |
| Expectations bullets | 60 chars | Each individual bullet |
| First name | Single name | No surnames or initials |
| Age | Specific number | Not a range |
| All paragraphs | 2-3 sentences | Consistent throughout |

## Quality Checklist

### Before Export

- [ ] Total pages ≤ 11
- [ ] Cover page complete with validation summary
- [ ] Evidence base page with interview breakdown
- [ ] 3-5 personas, each on new page
- [ ] All validation badges match interview counts
- [ ] All bullets ≤60 characters (verified)
- [ ] All paragraphs 2-3 sentences
- [ ] Current solutions specifically named
- [ ] Ages are specific numbers
- [ ] Consistent formatting throughout
- [ ] No placeholder text remaining
- [ ] Color coding applied correctly

### File Naming Convention

```
[CompanyName]_Vianeo_Personas_[Date].docx
```

Example: `TechStartup_Vianeo_Personas_2025-01.docx`
