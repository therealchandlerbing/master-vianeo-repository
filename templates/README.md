# Templates Directory

This directory contains reusable output templates for all VIANEO framework deliverables.

## Structure

### `/markdown/`
Markdown templates for:
- Executive briefs
- Assessment results
- Analysis reports
- Evidence logs
- Hypothesis tracking logs

### `/docx/`
DOCX format templates for:
- Professional reports
- Application forms
- Diagnostic comments
- Persona documents
- Assessment summaries

### `/html/`
HTML templates for:
- Interactive matrices (features-needs, needs qualification)
- Network visualizations
- Dashboard reports
- Print-optimized layouts

## Template Types

Templates are provided for each step that generates output:

- **Step 0**: Executive Brief (Markdown + DOCX)
- **Step 1**: Application Forms (360 SIS + CNEN formats)
- **Step 2**: 40Q Diagnostic (Assessment + Summary)
- **Step 3**: 29Q Market Maturity (3 documents)
- **Step 4**: Legitimacy Worksheet (Markdown + DOCX)
- **Step 5**: Needs/Requesters Analysis (4-file package)
- **Step 6**: Persona Development (Professional DOCX)
- **Step 7**: Needs Qualification (HTML Matrix + HTML Report)
- **Step 8**: Players & Influencers (2-page DOCX + Markdown)
- **Step 9**: Ecosystem Value Network (4 deliverables)
- **Step 10**: VIANEO Diagnostic (Markdown + DOCX)
- **Step 11**: Features-Needs Matrix (HTML + Analysis MD)

## Usage

All templates follow consistent formatting standards:
- Professional typography
- Semantic color coding
- Clear section hierarchy
- Quality checklists
- Evidence requirements

Templates will be populated from the vianeo-platform-tools repository.
