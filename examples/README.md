# Examples Directory

This directory contains sample completed assessments demonstrating the VIANEO framework in action.

## Structure

### `/by_stage/`
Examples organized by maturity stage:
- **IDEA**: Concept-stage projects with early validation
- **PROTOTYPE**: Projects with working prototypes and initial testing
- **PILOT**: Projects with pilot programs and customer feedback
- **EARLY_COMMERCIALIZATION**: Projects with initial revenue
- **GROWTH**: Scaling projects with proven business models

## Example Components

Each example includes:

### Complete Assessment Package
- Step 0: Executive Brief
- Step 2: 40-Question Diagnostic Results
- Step 3: 29-Question Market Maturity Assessment
- Selected deep-dive steps (4-9) as applicable
- Step 10: VIANEO Diagnostic Comment
- Step 11: Features-Needs Matrix (if applicable)

### Context Documentation
- Project background
- Industry/sector information
- Key challenges addressed
- Unique evaluation considerations

### Anonymized Data
All examples use anonymized or public information to protect confidentiality while demonstrating methodology.

## Learning Resources

Examples demonstrate:
- How to score different maturity stages appropriately
- Evidence requirements at each stage
- Common patterns and red flags
- Quality standards for deliverables
- Progression from validation to execution phases

## Usage

Use examples to:
- Learn the framework methodology
- Calibrate scoring standards
- Understand output quality expectations
- Train new evaluators
- Reference during actual assessments

Examples will be added from completed assessments, with appropriate anonymization.

## Example Categories

Planned examples covering:
